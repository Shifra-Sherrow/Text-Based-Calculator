Hello everyone!

This calculator evaluates a series of assignment operators provided as command-line input and prints the final values of the variables.
Press Return to calculate, Shift + Return to add a new line

Characters
The following characters are supported:
letters (uppercase & lowercase), digits, space, dot, (, ), =, *, /, %, +, -, $, _

Operators
The following operators are valid:
=, *=, /=, %=, +=, -=, +, -, *, /,%, **, ++, --

Variables
Variable names can be any valid JavaScript variable name: they can include letters, digits, dollar signs ($) and underscores (_). They can not start with digits.

Numbers
All real numbers are okay.

For example:
i = 0
j = ++i
x = i++ + 5
y = 5 + 3 * 10
i += y

Result:
i: 37, j: 1, x: 6, y: 35
